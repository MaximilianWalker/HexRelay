# Web App

Next.js TypeScript scaffold for the HexRelay web client.

## Runtime Position

- Primary product target is desktop local-first distribution.
- This app provides the UI layer used by desktop packaging and optional browser compatibility mode.
- API/realtime server logic remains in Rust services and is not bundled into browser runtime JS.

## Commands

- Install: `npm install`
- Run dev server: `npm run dev`
- Lint: `npm run lint`
- Test: `npm run test`
- Build: `npm run build`
